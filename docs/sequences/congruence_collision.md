# Congruence Collision Oscillator

> **Formula:** `a(n) = a(n-1) + dir * n; if a(n)%n is prime, a(n) %= n and dir flips`

## Description
Grows by n in current direction. Crashes to a(n) % n and flips direction whenever a(n) % n is prime.
