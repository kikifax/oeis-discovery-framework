# Thue-Morse Prime Walk

> **Formula:** `a(n) = a(n-1) + (-1)^popcount(a(n-1)) * p_n`

## Description
a(n) = a(n-1) + p_n if popcount(a(n-1)) is even, else a(n-1) - p_n. a(0)=0.
