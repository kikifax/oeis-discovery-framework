# Super-Prime Collision Oscillator

> **Formula:** `a(n) = a(n-1) + dir * p_k; dir flips and k=1 if a(n) is Super-Prime`

## Description
Moves by p_k in current direction. K increments every step, but resets to 1 and flips direction when a(n) hits a Super-Prime (prime whose index is also prime).
