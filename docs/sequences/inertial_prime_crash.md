# Inertial Prime Crash

> **Formula:** `a(n) = a(n-1) + dir * step; step resets and dir flips if a(n) is prime and n > last_n + 10`

## Description
Accelerates linearly in current direction. Crashes and flips direction if prime, but only after a mandatory 10-step acceleration period.
