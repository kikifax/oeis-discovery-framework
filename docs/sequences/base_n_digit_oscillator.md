# Base-N Digit Oscillator

> **Formula:** `a(n) = a(n-1) + n; if count(digits_base_n(a(n), 1)) is prime, a(n) = floor(sqrt(a(n)))`

## Description
a(n) = a(n-1) + n. If the count of '1's in the base-n representation of a(n) is prime, a(n) = sqrt(a(n)).
