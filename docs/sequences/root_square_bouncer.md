# Root-Square Bouncer

> **Formula:** `a(n) = a(n-1) + dir * floor(sqrt(n)); dir flips if is_square(a(n))`

## Description
a(n) = a(n-1) + dir * floor(sqrt(n)). The direction flips whenever a(n) is a perfect square. a(0)=2.
