# Mirror Prime Bouncer

> **Formula:** `a(n) = |a(n-1) + dir * p_n|; dir flips if prime or hit zero`

## Description
a(n) = a(n-1) + dir * p_n. If the result is negative, it mirrors at zero (absolute value) and direction becomes UP. Otherwise, if the result is prime, the direction flips. a(0)=0.
