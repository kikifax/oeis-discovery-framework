# Divisor-Balance Walk

> **Formula:** `a(n) = a(n-1) + dir * n; dir flips and n resets to 1 if sigma0(a(n)) == sigma0(n)`

## Description
Grows by current step size in current direction. Step size resets and direction flips if a(n) and n have the same number of divisors.
