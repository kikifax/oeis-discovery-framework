# Greedy Pioneer Walk

> **Formula:** `a(n) = a(n-1) + dir * s; s++ if a(n) is new; dir flips if is_square(s)`

## Description
Moves by summand 's' in current direction. s increments whenever a new number is discovered. Direction flips whenever the summand 's' is a perfect square. a(0)=0.
