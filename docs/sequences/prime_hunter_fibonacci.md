# Accelerated Prime Fibonacci

> **Formula:** `a(n) = a(n-1) + a(n-2) + m if is_prime(a(n-1)) else |a(n-1)-a(n-2)|`

## Description
a(n) = a(n-1) + a(n-2) + momentum if a(n-1) is prime, else |a(n-1) - a(n-2)|. Momentum increments during growth and resets on crashes.
